## Project Overview
This project delivers a complete end-to-end data analytics solution for a retail business,
covering data cleaning, database design, analysis, customer segmentation, and interactive
business dashboards.

## Tools & Technologies
- Excel (Data Cleaning & Preparation)
- SQL (Database & Business Queries)
- Python (EDA & Customer Segmentation)
- Power BI (Dashboard & Visualization)

## Business Objectives
- Analyze sales performance across stores and staff
- Identify high-value and at-risk customers
- Perform customer segmentation using RFM analysis
- Monitor inventory and sales KPIs
- Enable data-driven business decisions

## Project Workflow
1. Data cleaning and preprocessing in Excel
2. Relational database creation and querying using SQL
3. Exploratory Data Analysis and customer segmentation using Python
4. Interactive dashboard creation using Power BI
